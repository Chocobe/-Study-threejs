NOTICE

The completed versions of each class are located in the src -> note folder.
To check the lecture notes, copy the content of the corresponding JavaScript file, and paste it inside the index.js file to view them.

The icon creation, custom workspace, and space scene examples have separate C4D files available in
different versions: modeling, UV mapping, and baked versions. Each file is named following the pattern below:

Example:
    scene-1.c4d (Original version for icon creation modeling)
    scene-1-edit.c4d (Extended modeling and UV work version)
    scene-1-bake.c4d (Version with completed baking)

Please use these file names as a reference to locate and work with the respective versions of the C4D files.

The completed files for lectures 47 and 48 are located in the notes folder.
